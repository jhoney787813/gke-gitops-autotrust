Arquitectura GitOps sobre Kubernetes

Demostración práctica de despliegue continuo utilizando una filosofía declarativa. Sincronización automática de infraestructura mediante Argo CD y un clúster real en Killercoda.

Flujo GitOps

1. Git Push

El desarrollador sube cambios al repositorio GitHub (fuente de verdad).

2. Detect & Sync

Argo CD detecta el drift y reconcilia el estado deseado vs. el actual.

3. Deploy

Kubernetes aplica los manifiestos y despliega 3 réplicas del Pod.

4. Healthy

La aplicación queda disponible con self-healing y alta disponibilidad.
